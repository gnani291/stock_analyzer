📈 Stock Market Analyzer
Live Demo:https://stock-analyzer-966z.onrender.com/
A real-time stock market analysis web application built using Python, Streamlit, NumPy, and Yahoo Finance API. This app allows users to search for company stocks by name and instantly view stock insights, statistics, and graphical analysis.

🚀 Features


🔍 Search stocks using company names


📊 Real-time stock data fetching


📈 Interactive stock price visualization


🧮 NumPy-based stock analysis


📉 Moving Average calculation


💹 Profit/Loss percentage analysis


🌐 Web-based UI using Streamlit


☁️ Deployed on Render

⚡ Installation
Clone the repository:
git clone https://github.com/your-username/stock-market-analyzer.git
Move into the project folder:
cd stock-market-analyzer
Install dependencies:
pip install -r requirements.txt
Run the project:
streamlit run main.py

📊 How It Works


User enters company name


App converts it into stock ticker symbol


Yahoo Finance API fetches stock data


NumPy performs analysis calculations


Matplotlib generates graphs


Streamlit displays everything on a web interface
